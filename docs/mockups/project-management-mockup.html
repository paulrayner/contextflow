Design Notes

  • Sample projects: Cannot be deleted (no trash icon), but can be duplicated
  • Active project: Highlighted with blue border and "Active" badge
  • Hover actions: Duplicate and delete icons appear on hover
  • Inline rename: Double-click project name to edit (not shown in mockup)
  • Returning users: Welcome modal prioritizes their projects over samples
  • Empty state: If no user projects, show "New Project" button prominently
